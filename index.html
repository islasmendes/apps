Backup & dados

Salvo neste navegador, por mês. Exporte um backup para não perder nada.
Como funciona o armazenamento?
Tudo é salvo localmente (localStorage) no navegador de cada gestor — sem servidor e sem login. Cada mês é guardado separadamente: troque o mês no topo para ver/editar o período. Limpar o cache apaga os dados, por isso use o backup.

Fechamento Diário

Resumo do dia + parecer. Escolha o dia. Só o % é automático (Real ÷ Meta). Outras lojas entram quando o tempo real for ligado.
No Parecer, arraste a barra para o lado para ler tudo (o resto fica fixo). Meta/Real/Prev são digitados; o % se calcula sozinho. Tab avança para o próximo indicador à direita. Sábado e domingo ficam bloqueados — clique no 🔓 no dia para liberar. Em Puxada, em branco = não fez; selecione Sim só quando realizada. Cores do %: 0–50% · 51–78% · ≥79% (aceitável para não ir à reunião divisional).
