Backup & dados

Os dados ficam salvos neste navegador, por mês. Exporte um backup para não perder nada e para passar a base entre aparelhos.
Como funciona o armazenamento?
Tudo é salvo localmente (localStorage) no navegador de cada gestor — sem servidor e sem login. Cada mês é guardado separadamente: troque o mês no topo para ver/editar o período. Limpar o cache do navegador apaga os dados, por isso use o backup.
